# Stock Trading Platform - Java

This project is created as part of the CodeAlpha Java Programming Internship.

## Project Description
A console-based Java application that simulates a basic stock trading system.
Users can buy and sell stocks, view portfolio details, and check account balance.

## Features
- Buy and sell stocks
- View portfolio
- Track available balance
- Uses Object-Oriented Programming concepts

## Technologies Used
- Java
- OOP
- Collections (HashMap)

## How to Run
1. Compile the program:
   javac task2.java
2. Run the program:
   java task2
